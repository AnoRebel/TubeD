# TubeD

**A simple proof-of-concept youtube video and audio downloader**

Made with:

 - Nodejs
 - ytdl-core
 - Electron

To run this;
 
  - Clone the repo
  
	`git clone https://github.com/anorebel/TubeD.git`
 
  - Install dependencies
  
	  `npm install`
or
	  `yarn`
	  
 - Run:
 	- In development:
	 	`npm dev`
	 		or
	 	`yarn dev`
	 	
	 - In production:
	 	`npm start`
	 		or
	 	`yarn start`
